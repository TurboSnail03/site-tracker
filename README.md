# 🏗️ Site Tracker: Construction Expense Manager

A high-performance, mobile-first React application designed for real-time construction expense tracking. Built to be "site-ready" with a focus on low-friction data entry and eye-friendly aesthetics.

## 🚀 Key Features
- **PWA Ready:** Installable on iOS and Android for a native app experience.
- **Vintage Cream Mode:** High-contrast, warm-tinted light mode to reduce eye strain in outdoor/low-light environments.
- **Frictionless Logging:** One-tap logging directly from material cards.
- **Smart Search & Filter:** Instantly find transactions by material name or vendor.
- **Cloud Sync (Serverless):** Global drag-and-drop JSON backup system for cross-device data portability without needing a database account.
- **Mobile Native Polish:** Haptic feedback, anti-zoom inputs, and safe-area awareness for modern smartphones.

## 🛠️ Tech Stack
- **Frontend:** React.js (Vite)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Persistence:** Browser LocalStorage with JSON Export/Import

## 📦 Deployment Instructions
1. Run `npm install` to install dependencies.
2. Run `npm run build` to generate the production folder.
3. Deploy the `/dist` folder to Vercel, Netlify, or GitHub Pages.
